# Blog API 

## Описание проекта

Реализовать REST API для блога с функциональностью:
- Аутентификация пользователей (JWT)
- CRUD операции для постов
- Комментарии к постам
- Авторизация (только автор может редактировать/удалять свои посты и комментарии)

## Структура проекта

```
blog-api/
├── cmd/api/              # Точка входа приложения
│   └── main.go
├── internal/             # Внутренние пакеты приложения
│   ├── model/           # Модели данных
│   ├── handler/         # HTTP хендлеры
│   ├── service/         # Бизнес-логика
│   ├── repository/      # Работа с БД
│   └── middleware/      # HTTP middleware
├── pkg/                 # Переиспользуемые пакеты
│   ├── auth/           # JWT и пароли
│   └── database/       # Подключение к БД
├── migrations/         # SQL миграции
├── docker-compose.yml  # PostgreSQL и Adminer
├── .env.example        # Пример конфигурации
├── go.mod
└── README.md
```
## Функциональность

* Регистрация и аутентификация пользователей.
* Создание, чтение, обновление и удаление постов.
* Добавление комментариев к постам.
* Контроль доступа: только автор может редактировать/удалять свои посты.
* Пагинация для списка постов.
* Логирование запросов.
* Ограничение частоты запросов (rate limiting).

## Технологии

* Go 1.25+
* PostgreSQL
* JWT для аутентификации
* Chi — маршрутизатор HTTP
* Bcrypt — хеширование паролей
* Godotenv — загрузка переменных окружения
